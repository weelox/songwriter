# Songwriter Guide

En enkel och pedagogisk webbapp för ungdomar som vill skriva egna låttexter.

## Funktioner

- Steg-för-steg quiz med 4 startfrågor.
- Ett skrivflöde i låtstruktur:
  - Intro (valfritt)
  - Vers 1
  - Refräng
  - Vers 2
  - Outro (valfritt)
- Pedagogiska tips i varje skrivsteg.
- Slumpknapp som genererar startmeningar.
- Ljudfeedback (kan stängas av).
- Autosave i webbläsaren (`localStorage`).
- Import/export av projektfil (`.json`) för att pausa och återuppta senare.
- Export av färdig text som `.txt`.
- PDF-export via webbläsarens utskriftsfunktion.

## Kom igång

Öppna `index.html` i webbläsaren.
